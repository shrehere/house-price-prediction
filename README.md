# House Price Prediction

Predicts house prices using XGBoost with a R2 score of 81.7%.

## Overview
Explored Linear Regression, Random Forest, XGBoost on the dataset, iterating from base linear regression model to optimized XGBoost Model.

- Removing NaN rows and encoding categorically 
**Model 1- Linear Regression**
R2 score 11.5%
**Model 2- Random Forest**
R2 score 40.2%
**Model 3- XGBoost**
R2 score 41.6%

- Data Transformation (One Hot encoding, Target encoding and dropping redundant columns)
**Model 4- XGBoost**
R2 score 81.7%
- Adjusting model parameters
**Model 5- Random Forest**
R2 score 81.6%
